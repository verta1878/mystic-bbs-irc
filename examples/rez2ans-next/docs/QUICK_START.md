# Quick Start

## Convert an image

1. Start Rez2ANS Next.
2. Open an image or drag it into the application.
3. Choose the output dimensions and position the crop around the important subject.
4. Use the image and color controls to simplify the image for textmode.
5. Generate a preview and compare the result with the source.
6. Adjust settings or load a profile until the character shapes and colors are working for the piece.
7. Set an output directory if you want exports collected in one place.
8. Save ANSI, BIN, or a PNG preview.

Profiles let you save a successful setup and reuse it for similar artwork.
Ready-to-load examples are available in the [sample profiles directory](../profiles/README.md).

## Source-image guidance

ANSI conversion works best when the image already communicates clearly at a small size. Favor:

- a single clear subject
- strong silhouettes and thick outlines
- high contrast and large areas of color
- simple backgrounds and limited texture
- large, readable details

Photographs, fine textures, subtle gradients, small lettering, and crowded scenes usually need more preparation or manual cleanup after conversion.

## Export choices

- **ANSI**: terminal-ready textmode output.
- **BIN**: TheDraw BinaryText output for compatible editors and BBS workflows.
- **PNG**: a lossless image of the rendered ANSI preview, useful for sharing results online.

For a finished scene piece, use the conversion as a base and refine it in an ANSI editor: repair lettering, improve outlines, adjust colors, and add the details that make the piece yours.

For help creating source artwork with large shapes and a limited palette, see the [prompt guides](../prompts/README.md).
