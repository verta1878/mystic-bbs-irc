# RIPterm 1.54 — Original Freeware Distribution

Source: https://github.com/cgorringe/RIPterm154
Archived by: Carl Gorringe (cgorringe)
License: Freeware (originally distributed freely on BBSs)

This is the original MS-DOS RIPterm 1.54 freeware binary distribution,
originally created by TeleGrafix Communications, Inc. and freely
distributed on BBSs in the 1990s. Archived on GitHub by Carl Gorringe.

No license file was included in the original distribution or the
GitHub archive. The software was distributed as freeware.

## Contents

- `FONTS/` — 10 Borland BGI CHR vector fonts
- `ICONS/` — 212 ICN icon files + 4 MSK masks + 3 HIC files
- `RIPTERM.EXE` — Original MS-DOS RIPterm binary
- `RIPTERM.DOC` — Original documentation
- `README.1ST` — Original readme
- `WHATSNEW.DOC` — Original changelog
